# repaso-IPE-final

Web de repaso para **IPE II (Unidad 8)** con:

- Resumenes por tipo de empresa.
- Test interactivo de opcion multiple con correccion inmediata.

## Ejecutar en local

Como es una web estatica, basta con abrir `index.html` en el navegador.

## Publicar en GitHub Pages

1. Sube estos archivos a la rama `main`.
2. En GitHub, entra en `Settings` > `Pages`.
3. En `Build and deployment`, elige:
	- `Source`: `Deploy from a branch`
	- `Branch`: `main` y carpeta `/ (root)`
4. Guarda y espera la URL publica.

## Archivos

- `index.html`: estructura y contenido de la web.
- `styles.css`: estilos responsive.
- `script.js`: resumenes y logica del tipo test.
